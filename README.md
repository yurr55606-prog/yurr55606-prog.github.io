# 黑洞 · 五维作品空间

一个以“穿越黑洞 → 进入高维时间档案馆 → 探索作品”为叙事线的中文个人网站。

## 本地运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

作品文字集中在 `src/data.js`，可以直接替换为真实内容。

## 部署与资源保护

源工程保存在私有 GitHub 仓库，通过 Vercel 从 `main` 分支自动部署。构建时，图片、视频与角色素材会生成无语义哈希文件名；公开响应同时禁止搜索引擎收录素材并限制跨站引用。

浏览器必须下载正在展示的前端代码和媒体，因此这些措施用于隐藏源工程、原始命名与目录结构并提高批量复刻成本，不能阻止访客保存已显示的内容或截图。
